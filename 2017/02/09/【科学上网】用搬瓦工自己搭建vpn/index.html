【科学上网】用搬瓦工自己搭建vpn

发表于 2017-02-09 | 分类于 搭建vpn

废话不多说，直接进入搭建过程

准备

  • 钱
  • 钱
  • 钱

搭建过程

首先点击进入搬瓦工官网

点击右上角 register进行注册

其中Email adress（邮箱地址） 和Pasword(密码)和confirm Password（确认密码）要认真填写。 还有需要注意的是country 要填成China 其他随意填写》

下一步，注册好后重新登陆，然后点击servers-order new servers

出现下面页面

点order now
继续出现如下界面，按图中填写后，点击add to cart

接着按照指示的、付款即可。支持支付宝付款（手动滑稽）
购买成功后再进入首页—servers-my serves
点击图片中红色部分

接着记住ip adress

接着点击Shadowsocks Server 接着安装就好


将上图数据填入shadowsocks 即可成功翻墙~~~~
